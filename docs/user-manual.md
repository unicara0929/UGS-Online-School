## 1. はじめに

### UGSオンラインスクールとは

UGSオンラインスクールは、「学び → 実践 → 自立」を一体化したFP育成プラットフォームです。
教育コンテンツで金融知識を学び、FPエイドとして活動することで収益を得ることができます。

### ロール（会員種別）について

| ロール | 説明 |
|--------|------|
| **UGS会員（MEMBER）** | 基礎編の教育コンテンツで学習できます。FPエイドへの昇格条件を満たすと申請可能。 |
| **FPエイド（FP）** | 実践編・応用編の教育コンテンツにアクセス可能。報酬を得る活動ができます。 |
| **マネージャー（MANAGER）** | チームを管理できます。FPエイドとしての実績が必要です。 |
| **管理者（ADMIN）** | システム全体を管理できます。 |

### ログイン方法

1. `/login` にアクセス
2. 登録したメールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック
4. ダッシュボードに遷移します

## 2. 会員登録の流れ

> 新規会員は以下の流れで登録が完了します。動作確認時は各ステップが正常に動作するか確認してください。

### ステップ1: 仮登録

- `/register` で名前・メールアドレス・パスワードを入力

**確認ポイント:**
- [ ] 紹介コード入力欄があるか
- [ ] バリデーション（パスワード要件など）が動作するか

### ステップ2: メール認証

- 登録メールアドレスに認証リンクが送信されます

**確認ポイント:**
- [ ] 認証メールが届くか（迷惑メールフォルダも確認）
- [ ] 認証リンクをクリックして認証完了するか

### ステップ3: 決済（Stripe）

- `/checkout` で月額サブスクリプションの決済

**テスト用カード番号:**
```
4242 4242 4242 4242
有効期限: 任意の将来の日付
CVC: 任意の3桁
```

### ステップ4: 登録完了

- 決済完了後、自動的にダッシュボードに遷移し、会員番号が付与されます

## 3. ダッシュボードの使い方

ログイン後に表示されるダッシュボードから、各種機能にアクセスできます。

### サイドバーメニュー

左側のサイドバーから各機能にアクセスできます。ロールによって表示されるメニューが異なります。

| メニュー | 説明 |
|----------|------|
| 教育コンテンツ | コース・レッスンの視聴 |
| 資料 | PDFなどの資料ダウンロード |
| イベント | イベントへの参加登録 |
| 通知 | お知らせの確認 |

**確認ポイント:**
- [ ] サイドバーのメニューがロールに応じて表示されるか
- [ ] アクティブなメニューがハイライトされるか
- [ ] モバイル表示でハンバーガーメニューが動作するか
- [ ] ユーザー名とロールが正しく表示されるか

## 4. 教育コンテンツ（コース・レッスン）

### コースの種類

| レベル | コース名 | アクセス可能ロール |
|--------|----------|-------------------|
| 基礎編 | 所得を増やす | 全ロール |
| 実践編 | 生き方を豊かにする | FP以上 |
| 応用編 | スタートアップ支援 | FP以上 |

### 操作手順

1. サイドバーから「教育コンテンツ」をクリック
2. コース一覧から学習したいコースを選択
3. 「学習を始める」ボタンをクリック
4. 動画を視聴（90%以上で完了扱い）
5. 次のレッスンへ進む

**確認ポイント:**
- [ ] コース一覧が正しく表示されるか
- [ ] ロックされたコースに「FPエイド限定」と表示されるか
- [ ] 動画が再生されるか（Vimeo連携）
- [ ] 視聴進捗が保存されるか
- [ ] 90%以上視聴で「完了」になるか
- [ ] NEWバッジが新しいコースに表示されるか

## 5. イベント参加

### イベントの種類

| タイプ | 説明 |
|--------|------|
| オンライン | Zoom等でのオンライン開催 |
| オフライン | 会場での対面開催 |
| ハイブリッド | オンライン・オフライン両方 |

### 参加登録の手順

1. サイドバーから「イベント」をクリック
2. 参加したいイベントの「参加登録」ボタンをクリック
3. 有料イベントの場合は決済ページへ
4. 登録完了

### 出席確認方法

**方法1: 参加コード入力**
- イベント当日に告知される参加コードを入力して出席完了

**方法2: 録画視聴+アンケート**
- 録画を視聴 → 「視聴完了」をクリック → アンケート回答で出席完了

**確認ポイント:**
- [ ] イベント一覧が表示されるか
- [ ] 対象ロール以外のイベントが非表示になっているか
- [ ] 参加登録ができるか
- [ ] 有料イベントで決済ができるか
- [ ] 参加コード入力が動作するか

## 6. FPエイドへの昇格

> UGS会員がFPエイドに昇格するには、以下の条件をすべて満たす必要があります。

### 昇格条件（申請前）

| # | 条件 | URL |
|---|------|-----|
| 1 | LP面談の完了 | `/dashboard/lp-meeting/request` |
| 2 | 基礎テスト合格（80%以上） | `/dashboard/basic-test` |
| 3 | アンケート提出 | `/dashboard/survey` |

### 申請後の手続き

| # | 手続き | 備考 |
|---|--------|------|
| 4 | 管理者の承認を待つ | - |
| 5 | 連絡先情報の確認 | 電話番号・LINE IDの入力 |
| 6 | コンプライアンステスト合格（90%以上） | `/dashboard/compliance-test` |
| 7 | ガイダンス動画視聴 | `/dashboard/fp-onboarding` で90%以上視聴 |

**確認ポイント:**
- [ ] `/dashboard/promotion` で条件達成状況が表示されるか
- [ ] 条件を満たすと「FP昇格申請」ボタンが表示されるか
- [ ] 承認後にオンボーディングフローに遷移するか
- [ ] コンプライアンステストが動作するか

## 7. 紹介機能

### 紹介の仕組み

1. `/dashboard/referrals` で自分の紹介コードを確認
2. 紹介リンクをコピー（例: `/register?ref=ABC12345`）
3. 知人にリンクを共有
4. 知人が登録・決済完了すると紹介実績としてカウント

**確認ポイント:**
- [ ] 紹介コードが表示されるか
- [ ] 紹介リンクがコピーできるか
- [ ] 紹介実績（UGS会員紹介数、FPエイド紹介数）が表示されるか

## 8. 報酬管理（FP以上）

> FPエイド以上のロールで報酬管理機能を利用できます。

### 報酬の確認方法

1. サイドバーから「報酬管理」をクリック
2. 月別の報酬一覧を確認
3. ステータス（PENDING/CONFIRMED/PAID）を確認

### 口座情報の登録

`/dashboard/settings/bank-account` で振込先口座を登録してください。

> **注意:** ゆうちょ銀行の場合は、記号・番号から自動変換されます。

**確認ポイント:**
- [ ] 報酬一覧が表示されるか
- [ ] 報酬の内訳が確認できるか
- [ ] 口座情報が登録・更新できるか

## 12. 【管理者】ユーザー管理

### ユーザー一覧（`/dashboard/admin/users`）

- 名前、メール、会員番号で検索
- ロール別、ステータス別でフィルター
- ユーザー詳細ページへ遷移
- CSVエクスポート

### ロール変更

1. ユーザー一覧から対象ユーザーを選択
2. ロールのプルダウンから新しいロールを選択
3. 確認ダイアログで「変更」をクリック

> **注意:** FPからMEMBERに変更すると、コンプライアンステスト合格状態がリセットされます。

**確認ポイント:**
- [ ] ユーザー検索が動作するか
- [ ] フィルターが動作するか
- [ ] ロール変更が反映されるか
- [ ] CSVエクスポートができるか

## 13. 【管理者】イベント管理

### イベント作成

1. `/dashboard/admin/events` で「新規作成」をクリック
2. タイトル、日時、場所を入力
3. 対象ロールを選択（複数可）
4. 有料イベントの場合は価格を設定
5. サムネイル画像をアップロード（オプション）
6. 「作成」をクリック

### 出席確認の設定

| 項目 | 説明 |
|------|------|
| 参加コード | イベント当日に告知するコードを設定 |
| 録画URL | Vimeo URLを設定（後日視聴用） |
| アンケートURL | Google Formなどのリンクを設定 |

**確認ポイント:**
- [ ] イベント作成ができるか
- [ ] 対象ロール以外のユーザーにイベントが表示されないか
- [ ] 有料イベントの決済が動作するか
- [ ] 参加者一覧が確認できるか

## 17. FAQ・よくある質問

### Q: パスワードを忘れました

A: ログイン画面の「パスワードをお忘れの方」から再設定できます。登録メールアドレスにリセットリンクが送信されます。

### Q: 動画が再生されません

A: ブラウザのキャッシュをクリアするか、別のブラウザをお試しください。
推奨: Chrome、Safari、Firefox の最新版

### Q: 決済エラーが発生しました

A: カード情報を確認の上、再度お試しください。解決しない場合はお問い合わせフォームからご連絡ください。

### Q: 退会したい

A: `/dashboard/settings/account` の「退会申請」から手続きできます。なお、解約後も期間終了までサービスをご利用いただけます。
