# ブラウザコンソールエラーの確認手順

## 確認方法

### ステップ1: ブラウザの開発者ツールを開く

1. ブラウザでアプリケーションにアクセス
2. **F12** キーを押す（または右クリック → **検証**）
3. 開発者ツールが開きます

### ステップ2: Consoleタブを確認

1. 開発者ツールの上部にあるタブから **Console** を選択
2. 赤いエラーメッセージがないか確認
3. エラーメッセージをコピー

### ステップ3: Networkタブを確認

1. 開発者ツールの上部にあるタブから **Network** を選択
2. ページをリロード（F5）
3. 赤いエラー（4xx, 5xx）がないか確認
4. エラーが出ているリクエストをクリック
5. **Response** タブでエラーメッセージを確認

## よくあるエラー

### 1. クライアントサイドエラー
- `Application error: a client-side exception has occurred`
- `Uncaught Error`
- `TypeError`

### 2. APIエラー
- `Failed to fetch`
- `NetworkError`
- `500 Internal Server Error`
- `503 Service Unavailable`

### 3. 認証エラー
- `401 Unauthorized`
- `403 Forbidden`
- `ユーザーが見つかりません`

## 確認してほしいこと

1. **ブラウザのコンソールにエラーが表示されているか**
2. **エラーメッセージの内容**
3. **Networkタブでエラーが出ているリクエストがあるか**
4. **どのページでエラーが発生しているか**（ログイン、ダッシュボードなど）

エラーメッセージを共有していただければ、具体的な対処法を提案できます。
